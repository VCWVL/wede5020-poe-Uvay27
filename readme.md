## Project Overview
This project is a comprehensive website for **Checkpoint Gaming**, a South African retail gaming store.  
The website showcases products, allows customer enquiries, manages shopping cart, and provides multiple contact options.

The goal is to create a **professional, functional, and SEO-optimized** website using HTML, CSS, and JavaScript.

---

## Website Goals and Objectives
- Create an online presence for Checkpoint Gaming  
- Allow customers to browse products (consoles, games, accessories)  
- Provide service enquiry forms for products, volunteering, and sponsorship  
- Implement shopping cart functionality  
- Optimize for search engines (SEO)  
- Ensure mobile-responsive design  
- Provide excellent user experience with interactive elements  

---

## Key Features and Functionality

### Part 3 Enhancements:
- **Advanced Form Validation:** Real-time validation with error messages
- **AJAX Form Submission:** Simulated asynchronous form processing
- **SEO Optimization:** Meta tags, sitemap, robots.txt, semantic HTML
- **Interactive Elements:** Image lightbox, product search, accordions
- **Shopping Cart:** Local storage persistence with quantity management
- **Responsive Design:** Mobile-first approach with enhanced styling

### Core Pages:
- **Home Page** – Introduction and featured deals with gaming tips
- **About Page** – History, mission, vision with interactive accordions
- **Products Page** – Searchable product grid with lightbox gallery
- **Enquiry Page** – Service, product, and sponsorship enquiry forms
- **Cart Page** – Persistent shopping cart with checkout simulation
- **Contact Page** – Store locations and validated contact form
